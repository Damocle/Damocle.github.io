javascript数据类型 

javascript数据类型：基本类型 、 引用（对象）类型、

基本类型与引用类型

基本类型：  Undefined、String、Boolean、Null、Number  
       不可以给引用类型的变量添加属性和方法
引用类型：  --> 对象  引用类型的值是保存在内存中的对象 
       可以给引用类型的变量添加属性和方法

基本类型变量的复制 ： 
            相当于创建了一个该变量的副本，两个变量完全独立、都可以参与任何操作而互不影响
引用类型变量的复制 ： 
            同样会将存储在变量对象中的值复制一份放入新的变量的内存空间，
            但该值实际只是一个指针，指针指向存储在堆内存中的一个对象，   
            复制结束后两个变量实际上引用同一个对像，所以改变其中一个变量，会影响另一个

Js中所有函数的参数都是按值传递的
            -->  函数外部的值复制给函数内部的参数，就是从一个变量复制到另一个变量
            
            所以：基本类型的变量值传递如同基本类型的
 
// 基本类型复制与引用

var a = 123;
var  b  =  a;  // a === b -> true
function add(x) {
    return x + 1;
}

var c = add(a); // 124
console.log(a); // 123

// 引用类型

var person = new Object();
console.log(person.name);  // undefined

function setName(obj) {
    return obj.name = "Mr.ding";
}
setName(person);
console.log(person.name); // "Mr.ding"

隐式类型转换--发生隐式类型转的四种情况：

1、  数值运算：+、-、*、/、%

// + 运算时，其中一个为数值，另一个为字符串，则数值会被隐式转换为字符串
 var a = 10;
 var b = a + "10";
 console.log(a); // "1010"
// +以外的数值运算时 其中一个为数值，另一个为字符串，则字符串会被隐式转换为数值
 var a = 10;
 var b = "5";
 var c = a - b;
 console.log(c); // 5

 
2、 " . " 运算  将数值类型隐式转换为数值对像
3、 if语句
4、==

javscript类型识别：

1、typeof操作符
可以识别原始类型（Null除外）、
不能识别具体的对像类型（function除外）
typeof "a";  // string   
typeof null; // object
typeof function() {...} // function
var a = new Array();
console.log(typeof a); // object2、instanceof

能够判别内置对象类型
能够判别原始类型
能够判别自定义类型

3、object.prototype.toString.call(obj).slice(8, -1);
可以识别标准类型及内置对象类型
不能识别自定义对象类型

4、constructor
可以判别标准对象类型（Undefined/Null除外）
可以判倍内置对象类型
可以判别自定义对象类型
